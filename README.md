Water Reminder Pro is a simple and fast hydration reminder website that helps users stay healthy by tracking daily water intake.

## Features
- Daily water goal calculation
- Smart hydration reminders
- Intake tracking (125ml, 250ml, Custom)
- Progress indicator
- History and streak system
- Offline support
- Privacy focused (no server required)

## Tech Stack
- HTML5
- CSS3
- JavaScript
- LocalStorage
